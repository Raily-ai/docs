## What is Raily?

Raily is the **AI-Ready Content Access Layer** that helps publishers, enterprises, and content creators monetize their premium content and intellectual property in the GenAI era. We connect your data and IP to AI systems while ensuring you maintain control, visibility, and revenue.

## Get Started in Minutes

<Steps>
  <Step title="Create Your Account">
    Sign up at [raily.ai](https://raily.ai/contact) to get your API keys and access the dashboard.
  </Step>
  <Step title="Connect Your Content">
    Integrate your content sources using our SDKs or REST API. We support all major CMS platforms.
    ```bash
    npm install @raily/sdk
    ```
  </Step>
  <Step title="Define Access Policies">
    Set up policies to control how AI systems can access and use your content.
    ```javascript
    const policy = await raily.policies.create({
      name: "Premium Content Access",
      rules: [
        { action: "allow", conditions: { hasLicense: true } },
        { action: "deny", conditions: { default: true } }
      ]
    });
    ```
  </Step>
  <Step title="Start Earning">
    Monitor usage through the dashboard and watch as your content generates revenue.
  </Step>
</Steps>
