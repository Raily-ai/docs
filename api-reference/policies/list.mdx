---
title: "List Policies"
api: "GET /v1/policies"
description: "Retrieve a list of all access policies"
---

## Query Parameters

<ParamField query="limit" type="integer" default="20">
  Number of items to return (1-100)
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor
</ParamField>

## Response

<ResponseField name="object" type="string">
  Always `list`
</ResponseField>

<ResponseField name="data" type="array">
  Array of policy objects
</ResponseField>

<RequestExample>

```bash cURL
curl https://api.raily.ai/v1/policies \
  -H "Authorization: Bearer raily_sk_xxxxx"
```

```javascript JavaScript
const policies = await raily.policies.list();

policies.data.forEach(policy => {
  console.log(`${policy.name} (${policy.id})`);
});
```

```python Python
policies = raily.policies.list()

for policy in policies.data:
    print(f"{policy.name} ({policy.id})")
```

</RequestExample>
